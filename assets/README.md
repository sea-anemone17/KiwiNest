# 🥝 Dear Kiwi

다마고치형 공부 기록 사이트입니다.

## 실행 방법

1. 이 폴더를 압축 해제합니다.
2. `index.html`을 브라우저로 엽니다.
3. 동물, 이름, 테마, 성격을 고릅니다.
4. 공부를 기록하면 경험치가 오르고 로그가 저장됩니다.

## 현재 기능

- 동물 선택
- 이름 설정
- 테마 선택
- 성격 선택
- 공부 기록
- 경험치 / 레벨업
- 간식 티켓
- 딸기우유 주기
- 쓰다듬기 기록
- 성장 일지
- localStorage 저장
- JSON 백업
